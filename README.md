## 📖 About

**Loksewa MD** is a comprehensive mobile learning platform built for individuals preparing for the Public Service Commission (Loksewa) and other competitive examinations in Nepal.

The app brings together all the resources an aspirant needs — from video lectures and eBooks to official government notices and legal documents — in a single, easy-to-use application.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎓 **Video Lectures** | Online instruction and archived video classes from expert educators |
| 📚 **eBooks & PDFs** | Curated electronic publications and study materials |
| 📋 **Syllabi** | Official and up-to-date exam syllabi |
| 📢 **Exam Notices** | Live exam announcements from PSC Nepal |
| ⚖️ **Legal Documents** | Official documents sourced from Nepal Law Commission |
| 🔍 **Easy Search** | Quickly find any content across the entire library |
| 📡 **Regular Updates** | Content is continuously refreshed with new material |
